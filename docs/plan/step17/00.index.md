# DocuLight i18n 구현 계획 — 목차

## 개요

DocuLight의 모든 사용자 노출 문자열을 i18n 시스템으로 교체한다.
한국어(ko)와 영어(en) 두 locale을 지원하며, 시스템 언어 기반 자동 감지 + 영어 기본값.

## 범위

| 포함 | 제외 |
|------|------|
| 트레이 메뉴 (5개 항목) | MCP 도구 설명/응답 (API 영어 고정) |
| 설정 UI (16개 라벨/버튼/힌트) | 콘솔 로그 메시지 (`[doculight]` prefix) |
| 뷰어 UI (4개 툴팁 + 2개 패널 헤더) | 브랜드명 "DocuLight" |
| 뷰어 동적 메시지 (4개) | 벤더 라이브러리 (src/renderer/lib/) |
| 설정 동적 메시지 (8개) | |
| 파일 연결 메시지 (12개) | |
| 창 관리 에러 메시지 (6개) | |

**총 i18n 대상: ~55개 문자열**

```
Phase 1 (인프라) ──▶ Phase 2 (Main Process) ──▶ Phase 3 (Renderer)
```
