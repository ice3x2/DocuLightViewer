# Phase 6 검증: 탭 기반 다중 문서 뷰

## 완료 체크리스트

- [ ] tab-manager.js 모듈 로드 및 init 정상
- [ ] 사이드바 클릭으로 새 탭 생성 (enableTabs: true)
- [ ] 탭 전환 시 상태 보존 (scrollTop, renderedHtml, 사이드바)
- [ ] 탭 닫기 → 이전 탭으로 전환

- [ ] 1개 탭일 때 탭 바 숨김
