# DocuLight Viewer v0.10.8 Release Notes

## Improvements
- **Removed SSE connection limit** -- The MCP HTTP server no longer restricts simultaneous SSE connections (previously capped at 5), allowing unlimited parallel agent access.
- **Removed request body timeout** -- The 30-second body read timeout has been removed, preventing unnecessary request failures for large payloads.
- **Removed content size limit** -- The 10 MB request body size cap has been removed, enabling transfer of arbitrarily large markdown documents via MCP.

# DocuLight Viewer v0.10.8 릴리즈 노트

## 개선 사항
- **SSE 연결 제한 제거** -- MCP HTTP 서버의 동시 SSE 접속 제한(기존 5개)을 제거하여 다수의 에이전트가 제한 없이 동시 접속 가능.
- **요청 본문 타임아웃 제거** -- 30초 본문 읽기 타임아웃을 제거하여 대용량 페이로드 전송 시 불필요한 요청 실패 방지.
- **콘텐츠 크기 제한 제거** -- 10 MB 요청 본문 크기 제한을 제거하여 MCP를 통한 대용량 마크다운 문서 전송 지원.
