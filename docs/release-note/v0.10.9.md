# DocuLight Viewer v0.10.9 Release Notes

## Bug Fixes
- **MCP port self-detection in settings** -- Fixed a bug where the settings screen incorrectly reported the current MCP server port as "in use", since the availability check was detecting DocuLight's own HTTP server.

# DocuLight Viewer v0.10.9 릴리즈 노트

## 버그 수정
- **설정 화면 MCP 포트 자기 감지 오류 수정** -- 설정에서 현재 MCP 서버 포트를 "사용 중"으로 잘못 표시하던 버그 수정. 포트 가용성 검사 시 자기 자신의 HTTP 서버를 오판하지 않도록 개선.
